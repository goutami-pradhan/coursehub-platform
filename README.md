# CourseHub - Online Course Enrollment System

## Overview

CourseHub is a full-stack web application where administrators can create and manage courses, and students can browse and enroll in courses.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Role-Based Authorization

### Admin
- Create Course
- Update Course
- Delete Course
- View All Courses
- Manage Student Enrollments

### Student
- Register/Login
- View Courses
- Enroll in Courses
- View Enrollment Status

## Tech Stack

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcryptjs

### Frontend
- React
- React Router
- Axios

## Project Structure

```
backend/
frontend/
```

## Run Backend

```bash
cd backend
npm install
npm run dev
```

## Run Frontend

```bash
cd frontend
npm install
npm start
```
